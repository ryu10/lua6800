# EDMON02

EDMON02 は 1981 年に発売された、MC6802 ベースのシングルボードコンピュータ Eaglet 02 用のモニタプログラムです。
「月刊 I/O」1981 年 10 月号に全回路図およびモニタプログラム操作マニュアルが掲載されました。

ここではオリジナルの PIA シリアルを ACIA に置き換えたバージョンをエミュレートします。


## 起動

以下のコマンドで起動します。

```bash
./run-m6800.sh edmon02/edmon02.bin
```

Edmon02 プロンプト `$` が表示されます。

終了するには `Ctrl-a k` を押し、プロンプトに対して `y` を押してください。

注:

EDMON02 バイナリは実機どおり `$F800-$FFFF` に配置されます。64k バイナリファイルは前方 62kB ぶん 0xff パディングされています。

## EDMON02 コマンド

下記が EDMON02 コマンドリファレンスからの引用です。

- M:  メモリ・イグザミンおよびチェンジ
- J or G: ユーザ・プログラム・スタート
- L: カセット・テープからロード
- S: カセット・テープにセーブ
- B: ブレーク・ポイント設定
- C: ブレーク・ポイント・クリア
- T: ワンステップ・トレース
- R: レジスタ表示
- Y: S1-S9 紙テープロード

SBC6800 Lua では以下のように実装されています。

`M <HEX4>` : 入力したアドレスのメモリ内容が表示されます。さらに続けて以下のサブコマンドが入力できます。

 - `<SPC><HEX2>` : そのアドレスに 1 バイトの値を変更
 - `/` : 次のアドレスを表示
 - `.<HEX4>` : 新しいアドレスに移動
 - `<CR>` : コマンド終了

入力パースがあまり良くありません。スペースのあとは必ず有効な HEX 数字（0-9,A-F）を入力してください。壊れた場合は `.<HEX4>` を入力するとやり直せる場合が多いです。

`J <HEX4>` : 入力したアドレスからユーザプログラムを開始します。

`L`/`S` : 実装されていません。

`B <HEX4>` : 入力したアドレスにブレークポイントを設定します。ブレークポイントは最大 5 個までです。

`R` : レジスタ内容を表示します CCR, B, A, X, PC, SP の順で表示されます（スタックに積まれる順）。

`Y` : S19 ファイルをロードします。適当なウェイトを挟みつつペーストしてください。こわれたときは `S9` を入力して中断できることが多いです。

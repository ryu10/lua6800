# lua6800

## これはなに？

- MC6800 SBC エミュレータ
- 6800/6802 モニタ開発用
- cpu エミュコードは moon6800 を使用

## 現状機能

- メモリモデルを追加
- acia 機能を追加

## 問題点

- mac で動いて ubuntu で動かない
- pty / socat 経由の一文字ノンブロッキング送受信（acia 実装）が ubuntu に限ってうまくいかない
- Claude は moon6800 / mikbug のせいにしたがるがこのふたつは mac で動作しているのでこれらの修正提案は棄却すること
